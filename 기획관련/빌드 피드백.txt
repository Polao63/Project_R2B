갈고리 사용
캐릭터 시선방향
=> 갈고리 사용 키 입력 시 캐릭터의 시선 방향이 갈고리가 나가는 방향이면 자연스러울 것 같습니다. v
갈고리 사용 중 이동 모션
=> 이동을 하면서 갈고리를 사용하면 갈고리 사용 중 이동 모션이 재생되는 점.
갈고리 후 자동 점프
=> 갈고리 후 자동 점프 => 그 뒤에 점프 키 입력 시 공중점프 모션으로 나오지 않거나 늦게 나오는 점. 
    공중점프 모션이 땅에 닿기 전에 안 됐을 경우 땅에서 처음 점프 시 공중점프 모션으로 재생되는 점.
